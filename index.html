1-8 找回密码接口

接口调用请求说明:

说明
协议 http
http请求方式 POST

数据格式 无

请求参数说明:

参数 是否必须 说明
mobile 是 手机
password 是 密码
confirmPassword 是 确认密码
verificationCode 是 手机验证码

返回数据说明:

参数 说明
code 成功：200
出错：-xx
msg 消息提示
obj 对象
