not null 和 default

2017年07月29日 |
数据库相关/ mysql

没有设置default

  • 当插入数据时，没有操作某一字段，并且该字段为null且没有defualt值时，mysql会自动出入null值；

  • 当该字段为not null且没有defualt值时，会报错

有设置default

若有设置default值并且插入数据时没有操作该字段，不管该字段是否设置not null 还是null，mysql统一设置default的值
